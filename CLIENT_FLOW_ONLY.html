Platform Overview

AngelX is a secure cryptocurrency exchange platform that enables users to:

  • Deposit fiat currency (INR) to receive cryptocurrency (USDT)
  • Withdraw cryptocurrency (USDT) to personal wallets
  • Exchange cryptocurrency (USDT) for fiat currency (INR)
  • Track all transactions and account balances

The platform operates on a dual-role system with distinct permissions for regular users and administrators.

User Roles

Regular Users

Regular users can:

  • Register and log in to the platform
  • Deposit INR to receive USDT
  • Withdraw USDT to external wallets
  • Exchange USDT for INR
  • Manage wallet addresses and bank accounts
  • View transaction history and account balance

Administrators

Administrators can:

  • Manage user accounts and balances
  • Create and update deposit methods
  • Set and update exchange rates
  • Verify and approve deposit transactions
  • Process withdrawal requests
  • Handle exchange transactions
  • View platform analytics and statistics

Core Features

1. Deposit Management

Users can deposit fiat currency (INR) which gets converted to cryptocurrency (USDT) based on the current exchange rate.

2. Withdrawal Management

Users can withdraw their USDT balance to external cryptocurrency wallets.

3. Exchange Services

Users can exchange their USDT balance for fiat currency (INR) which gets transferred to their bank accounts.

4. Wallet Management

Users can save and manage multiple cryptocurrency wallet addresses.

5. Bank Account Management

Users can save and manage multiple bank accounts for receiving fiat currency.

6. Transaction History

Users can view their complete transaction history including deposits, withdrawals, and exchanges.

Transaction Flows

User Registration & Authentication

User Registration Process:

  1. User provides a 10-digit phone number
  2. System sends an OTP for verification
  3. User verifies the OTP
  4. System creates the account

Authentication Process:

  1. User provides their phone number
  2. System sends an OTP
  3. User provides the OTP
  4. System authenticates the user and grants access

Deposit Process

The deposit process allows users to add funds to their account by depositing fiat currency (INR).

Step-by-Step Flow:

1. View Deposit Methods

  • User can see all available payment methods for deposits

2. Create Deposit Request

  • User selects a payment method and specifies INR amount to deposit
  • System creates a deposit request and provides payment details

3. Make Payment

  • User makes payment using the provided method details
  • User obtains a transaction ID (TXID) from the payment platform

4. Submit Transaction ID

  • User submits the transaction ID to the system
  • System marks the deposit as awaiting verification

5. Admin Verification

  • Admin reviews the pending deposit
  • Admin verifies the transaction by checking payment details
  • Admin approves or rejects the deposit

6. Balance Crediting

  • If approved, system converts INR to USDT using the current exchange rate
  • System adds the USDT equivalent to the user's balance

Key Points:

  • Users deposit fiat currency (INR) and receive cryptocurrency (USDT)
  • Balance is credited only after admin approval with proper conversion
  • Transaction IDs are required for verification
  • Deposit methods are managed by admins

Withdrawal Process

The withdrawal process allows users to transfer cryptocurrency from their account to external wallets.

Step-by-Step Flow:

1. Manage Wallets

  • User can add and save wallet addresses
  • User can view their saved wallet addresses

2. Create Withdrawal Request

  • User selects a wallet and specifies USDT amount to withdraw
  • User confirms the request with an OTP
  • System immediately deducts USDT amount from user balance
  • System creates a withdrawal request for admin processing

3. Admin Processing

  • Admin reviews the withdrawal request
  • Admin processes the withdrawal by sending cryptocurrency to user's wallet
  • Admin marks the withdrawal as completed
  • System creates a transaction record for audit purposes

Key Points:

  • Balance is deducted immediately when withdrawal is created
  • Status changes don't affect balance (balance already deducted)
  • Transaction records are for audit purposes only

Exchange Process

The exchange process allows users to convert cryptocurrency to fiat currency.

Step-by-Step Flow:

1. Manage Bank Accounts

  • User can add and save bank account details
  • User can view their saved bank accounts

2. Check Exchange Rate

  • User can check the current exchange rate
  • Admins can update rates as needed

3. Create Exchange Request

  • User selects a bank account and specifies USDT amount to exchange
  • System immediately deducts USDT from user balance
  • System creates an exchange request with INR equivalent

4. Admin Processing

  • Admin reviews the exchange request
  • Admin processes the exchange by transferring INR to user's bank account
  • Admin marks the exchange as completed
  • User receives fiat currency in their bank account

Key Points:

  • Exchange rate is used to calculate INR equivalent
  • USDT balance is deducted immediately upon request
  • Fiat transfer is processed manually by admin

Balance Management

Currency System

  • All user balances are stored in USDT (Tether cryptocurrency)
  • Deposits in INR are converted to USDT using the exchange rate
  • Withdrawals and exchanges use USDT balance directly

Balance Update Logic

Deposit Approval:

  1. Admin approves deposit
  2. System converts INR amount to USDT using current exchange rate
  3. System adds USDT equivalent to user balance

Withdrawal Creation:

  1. User creates withdrawal request
  2. System immediately deducts USDT amount from user balance
  3. Status changes do not affect balance (already deducted)

Exchange Creation:

  1. User creates exchange request
  2. System immediately deducts USDT amount from user balance
  3. Status changes do not affect balance (already deducted)

Important Balance Principles:

  1. Deposit: Balance credited only after admin approval
  2. Withdrawal: Balance deducted immediately upon request creation
  3. Exchange: USDT deducted immediately upon request creation
  4. Transaction Records: Audit trail only, no balance impact

Transaction Verification

Platform Analytics

  • View dashboard with key metrics
  • Monitor transaction volumes
  • Track user registration trends
